Dette var en kort opgave vi havde fået for derhjemme, hvor vi skulle se og nedskrive notater. Fremadrette har vi arbejdet med programmet til at lave videoer.
Link til videoen vi skulle lære fra: https://www.lynda.com/Premiere-Pro-tutorials/Adding-B-roll-supplement-your-interviews/371692/413954-4.html Chapter 2.

Her er de shortcuts jeg har fundet;

I to mark an in point

L double click = double speed on the video

O to make an out point

K = pause the video

Ctrl + shift + space bar = plays the video, marked between your in point and out point

Sub clip = ctrl + U

Sub Clip = Command + U

+ = zoom in

- = zoom out

Shift + Drag snaps the videos

Up arrow key = goes backwards

Down arrow key = goes forward

, = insert

Når man indsætter lyd og video, skal man have dem begge markeret, ellers kommer de ikke med.

Ctrl + alt + drag = kan trække et klip i dine timeline (swap shots)

’ = extract a clip mark in an in point and out point

A = track select forward

Shift + a = track select backwards

Space bar = pause / play

X = mark a clip

Ctrl + shift + x = clear in and out points and trim tool

When using trim tool, it gives a “gap” between the content, right click in the gap and press ripple delete to remove it.

B = Ripple Tool, will do the previous right away, trim without a gap.

N = rolling edit tool, change the length difference between 2 videos. Shorten one to make one longer, without touching other layers on the entire video.

Ctrl + click + keyframe

Shift + d = apply a default transition.
